## Nombre del proyecto:
ProyectoPhp

## Descripción:
Este proyecto es un ejemplo básico desarrollado en **PHP** que muestra en pantalla el mensaje: "Hola mundo, soy Diego Ara".

El objetivo del proyecto es demostrar el funcionamiento básico de PHP ejecutándose en un servidor local en Linux.

## Prerrequisitos:

Para ejecutar este proyecto necesitas instalar:

- Linux / Ubuntu
- Apache2: sudo apt install apache2
- PHP: sudo apt install php

## Ejecución:
- cd /var/www/html
- http://localhost/index.php
